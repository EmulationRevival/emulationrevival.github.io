# Overview

Xbox Dev Mode allows developer applications and emulators to run on Xbox One and Xbox Series consoles. Activation requires a Microsoft Developer account. Individual developer accounts are now **free**.

# Requirements

* Xbox One or Xbox Series console  
* Microsoft account  
* Internet connection  
* Windows 10/11 PC (recommended)

# Create Microsoft Developer Account

1. Open the developer registration page:

   https://developer.microsoft.com/store/register

2. Sign in using your **Microsoft account**.

3. Click **Create a developer account**.

4. Select:

   * **Account Type:** Individual  
   * **Country/Region:** Your real location

5. Complete the required profile information.

6. Accept the developer agreement.

7. Submit the registration.

The developer account activates immediately.

# Activate Dev Mode on Xbox

1. Open **Microsoft Store** on the Xbox.

2. Search for **Xbox Dev Mode**.

3. Install and launch the app.

4. The app displays:

   * A **Device Activation Code**
   * A **website address**

5. On a PC or phone open:

   https://aka.ms/activatexbox

6. Sign in using the **same Microsoft account used to create the developer account**.

7. Enter the **activation code** displayed on the Xbox.

8. Confirm activation.

The console will reboot into **Developer Mode**.

# Allocate Dev Storage

After the first boot into Dev Mode:

1. Open **Manage Dev Storage**.
2. Allocate **at least 5GB**.
3. Restart the console.

Dev Mode is now ready.

# Important Notes

* Switching between **Retail Mode and Dev Mode** removes:
  * Quick Resume states
  * Local captures

* If enrolled in **Xbox Update Preview**, leave it before enabling Dev Mode.

Steps:

1. Open **Xbox Insider Hub**
2. Go to **Previews**
3. Select **Xbox Update Preview**
4. Choose **Manage > Leave Preview**
5. Update the console
