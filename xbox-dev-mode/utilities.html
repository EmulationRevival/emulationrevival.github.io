A collection of essential tools to streamline your Xbox Developer Mode experience, from managing files to formatting drives.

Console Media USB Tool

Format a USB drive to NTFS and add permissions for UWP apps.
  • Developer: SvenGDK
  • Compatibility: Windows OS
  • Requires: External USB drive.
Download Watch Tutorial

Durango FTP

FTP Server for Xbox One/Series in Dev Mode.
  • Developer: Dantes-Dungeon
  • Compatibility: Xbox One & Series S|X
  • Note: FTP access is currently unreliable and may not work for all users.
Download Watch Tutorial

Oberon Remote

Remote input for Xbox consoles, use any controller with your Xbox.
  • Developer: SamsidParty
  • Compatibility: Xbox One & Series S|X
  • Requires: A separate device running Android, Windows, macOS or Linux.
Download Setup Guide

Safe Exit

Exit from dev mode safely.
  • Developer: Dantes-Dungeon
  • Compatibility: Xbox One & Series S|X
  • Note: This app is likely redundant now, as dev mode defaults to keeping sideloaded apps on exit.
