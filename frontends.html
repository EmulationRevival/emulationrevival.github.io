LaunchPass

LaunchPass is a frontend that is capable of launching ROMs in most emulators on Xbox consoles.

  • LaunchPass
  • Dependencies
Tutorial
  • Developer: Misunderstood-Wookiee & DanP
  • Compatibility: Xbox One & Xbox Series S|X
  • Requirements:
    • Install all 4 dependencies
    • Emulators: From a list of supported emulators
    • External storage: Used for setting up content library
    • A computer: For setting up a content library with LaunchBox

Retro Pass

Retro Pass is a simple frontend for emulators running on Xbox.

Download Tutorial
  • Developer: Retropassdev
  • Compatibility: Xbox One & Xbox Series S|X
  • Requirements:
    • Install all 5 (x64) dependencies
    • Emulators: From a list of supported emulators
    • External storage: Used for setting up content library
    • A computer: For setting up a content library with either:
      • LaunchBox
      • EmulationStation
